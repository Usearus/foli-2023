Terminal Commands:

npm start  -  Launch react environment
npm list   -  List out all packages

cd  -  go forwards in directory
cd..   -  Navigate to previous folder

npm error on new device? Open PowerShell as admin and type:
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Unrestricted

After grabbing an existing project:
npm install  -  Install your modules from the project
Copy over hidden files aka Environment keys from other device

If it has been a while since coding:
npm update -g  -  update all of your packages or check if they need to be updated.
